# 课程三：AI Agent 链上交易实战课

AI 正在学会花钱。Crypto 钱包让 AI Agent 可以自主行动。

这门课教你用 Agent 框架构建能操作链上资产的 AI Agent。

## 进阶内容

- 深入 Agent 框架核心：Character、Memory、Actions、Providers、MCP、Sub-agents、Skills、Plugins。
- 实现完整链上交易能力：钱包管理、DEX 对接、价格监控、风控。
- 集成付费 API（复用课程二的 x402）。

前置：课程一 + 课程二（推荐，非必需）。

## Part 1：背景知识 - AI Agent 基础（5 讲）

1. AI Agent 是什么：Agent vs Bot，为什么 Agent 能做交易。
2. 链上交易回顾：钱包、交易签名、Gas 费用。
3. DeFi 交互入门：DEX Swap、借贷操作、核心接口。
4. LLM 基础：Prompt 工程基础，让 AI 理解交易意图。
5. Agent 框架选型：ElizaOS vs LangChain vs Claude Agent SDK。

## Part 2：进阶技术 - ElizaOS 开发（12 讲）

模块 A：ElizaOS 框架精通

6. ElizaOS 架构解析：Character、Memory、Actions、Providers。
7. 创建 Agent 角色：人格设定、行为边界、风险偏好。
8. 记忆系统：短期记忆与长期记忆。
9. 自定义 Actions：Swap/Transfer/Approve 等链上操作。

模块 B：链上交易能力

10. 钱包集成：私钥管理、多链钱包、安全实践。
11. DEX 集成：Uniswap V3 / 1inch 交易执行。
12. 价格数据获取：API vs 预言机，实时监控。
13. 交易策略实现：网格、均值回归、止盈止损。

模块 C：运行与优化

14. 风险管理：仓位控制、单笔限额、紧急停止。
15. 日志与监控：交易记录、收益统计、异常告警。
16. Gas 优化：动态 Gas 调整、交易时机选择。
17. x402 集成：让 Agent 能调用付费 API。

## Part 3：Guided Project - DeFi 自动化助手（6 讲）

项目：稳定币收益聚合 Agent - 自动比较收益率并调仓。

用户输入：
- “帮我把 USDC 放到收益最高的地方。”

Agent 执行：
1. 查询 Aave/Compound/Yearn 的 USDC 收益率。
2. 比较后选择最优协议。
3. 执行 deposit 交易。
4. 定期检查，收益率变化时自动调仓。

18. 项目设计：需求分析、架构设计、支持协议列表。
19. Agent 核心开发：角色 + 意图解析 + 记忆系统。
20. 收益查询实现：多协议收益率 API 集成。
21. 交易 Actions：Deposit/Withdraw/Swap Actions 开发。
22. 测试调试：测试网验证、边界情况处理。
23. 部署运行：服务器部署、监控配置、收益追踪。

## 交付物

- 可运行的 ElizaOS AI Agent。
- 稳定币收益策略插件。
- 链上交易 Actions 模块。
- 监控面板与告警配置。
- 部署文档与安全说明。
- 可扩展的 Agent 模板。
